# FastMCP Crash Course

Welcome to the FastMCP Crash Course! This repository contains a structured set of lessons to help you master the FastMCP framework for building Model Context Protocol (MCP) servers.

## Lessons

1. [Lesson 1: Introduction to FastMCP](./lesson1_intro.md) - Learn the basics and set up your first server.
2. [Lesson 2: Tools and Resources](./lesson2_tools_resources.md) - Explore how to give LLMs capabilities and data.
3. [Lesson 3: Prompts and Context](./lesson3_prompts_context.md) - Master message templates and request metadata.

## Getting Started

To run the examples in this course, you'll need Python 3.10+ and the `fastmcp` package:

```bash
pip install fastmcp
```

## About FastMCP

FastMCP is the standard framework for building MCP applications. It provides a high-level, Pythonic interface for building, managing, and interacting with MCP servers.
